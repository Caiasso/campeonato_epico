# Campeonato épico de Pedra-Papel-Tesoura

##### Desenvolvido por Guilherme Henrique Caiasso e Mell Amisa Matsuda

## Descrição

O presente projeto é um campeonato do jogo Pedra-Papel-Tesoura utilizando a linguagem de programação Python. A dinâmica do campeonato é baseada em rodadas, onde a ordem dos jogadores é sorteada a cada rodada para competirem em pares. O projeto utiliza uma planilha chamada ```campeonato.csv```, que contém informações sobre os jogadores e as jogadas disponíveis. O caompeonato é projetado para funcionar em ambientes como Jupyter Notebook com Anaconda ou diretamente em uma máquina virtual no Google Colab.

Trabalho realizado para a disciplina de Tópicos Especiais em Programação I.

## Instruções

### 1. Ambiente de Execução:
Se estiver usando o Jupyter Notebook com Anaconda, é necessário ajustar o diretório para o correto funcionamento do código. Certifique-se de definir corretamente o diretório onde o arquivo CSV está localizado.

```
tabela = pd.read_csv('caminho/diretorio/arquivo.csv')
```
Se estiver usando o Google Colab, será necessário adicionar a base de conhecimento nos ```Arquivos``` da plataforma.

![img_colab.jpg](https://raw.githubusercontent.com/Caiasso/torneio_epico/main/img_colab.jpg)

### 2. Arquivo do campeonato:
O arquivo CSV contém informações sobre os jogadores e as jogadas disponíveis entre ```pedra```, ```papel``` e ```tesoura```.

### 3. Execução:
Execute o código do campeonato no ambiente escolhido (Jupyter Notebook ou Google Colab).
